Command Library

Explore HumanGPT's extensive command system

Bot Configuration

  • /enable_bot Activate HumanGPT in a channel
  • /disable_bot Deactivate in a channel
  • /settings Configure bot settings
  • /prefix Set custom command prefix

AI Features

  • /free_will Toggle random conversation joining
  • /generate_image Create AI artwork from text
  • /ask Ask HumanGPT any question
  • /chat_mode Set conversation style

Server Management

  • /welcome_message Set AI welcome messages
  • /auto_role Configure automatic roles
  • /server_stats View server analytics
  • /backup Create server backup

Fun & Entertainment

  • /meme Generate random memes
  • /trivia Start a trivia game
  • /joke Tell a joke
  • /quote Share inspirational quotes

Utility

  • /help Display all commands
  • /ping Check bot latency
  • /status View bot status
  • /user_info Get user information

Moderation

  • /warn Issue warning to user
  • /mute Temporarily mute user
  • /kick Remove user from server
  • /ban Permanently ban user
