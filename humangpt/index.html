Command Garden

Explore HumanGPT's extensive command ecosystem

Channel Management

  • /enable_bot Activate HumanGPT in a channel
  • /disable_bot Deactivate in a channel
  • /channel_settings Configure channel preferences
  • /auto_join_setup Set up automatic joining

AI & Conversation

  • /free_will Control random chatting
  • /generate_image Create AI artwork from text
  • /conversation_style Set chat behavior style
  • /ai_personality Adjust AI personality traits

Server Configuration

  • /welcome_message Set AI welcome messages
  • /moderation_settings Configure auto-moderation
  • /server_stats View server analytics
  • /role_management Manage user roles

Fun & Entertainment

  • /trivia Start a trivia game
  • /story_time Generate collaborative stories
  • /joke Tell a joke
  • /quote Share inspirational quotes

Utility Commands

  • /help Display all commands
  • /ping Check bot latency
  • /status View bot status
  • /user_info Get user information

Moderation Tools

  • /mute Temporarily mute a user

  • /ban Permanently ban a user
