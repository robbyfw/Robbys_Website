Privacy Policy – HumanGPT Discord Bot

Owned & Developed by Muhammad Asim

Last Updated: 18 December 2025

1. Introduction

This Privacy Policy explains how the Discord bot HumanGPT collects, stores, and uses data. By using the bot, you agree to this policy and to Discord’s Terms of Service.

2. Data We Collect

HumanGPT stores only the minimum data required for its functions, in JSON files:

  • Channel IDs – used for welcome messages and server configuration.
  • User IDs – used for AFK system commands.

No messages, DMs, passwords, or private information are stored.

3. Data We Do NOT Collect

HumanGPT does NOT collect:

  • Message content
  • Passwords or tokens
  • Payment or billing information
  • Email addresses (except optional contact forms)
  • IP addresses or physical location
  • Data from users under 13
  • NSFW content

4. How Data is Used

Data is used only to:

  • Provide and manage bot features
  • Save server configuration settings
  • Support the AFK system and welcome messages
  • Improve user experience

No user data is sent to AI providers. HumanGPT uses API keys only to communicate with Groq API and does not share or expose user data.

5. Analytics

We use Good Analytics to monitor bot performance and website traffic. No personal user data is sent to analytics services.

6. Data Access

Only the owner and authorized developers may access stored data when necessary for support or maintenance. HumanGPT data is never sold or shared with third parties.

7. Data Deletion

Users and server owners may request deletion of their data at any time. To do this, contact us via Discord or email (see Contact section).

8. Data Security

All data is stored securely in JSON files with encryption. We also use HTTPS for server communication. While we take security measures, no system is 100% secure, and you use the bot at your own risk.

9. Children’s Privacy

HumanGPT does not knowingly collect data from children under 13. Users must be 13+ to use the bot in compliance with Discord rules.

10. Changes to Policy

We may update this Privacy Policy at any time. Continued use of HumanGPT indicates acceptance of any changes.
